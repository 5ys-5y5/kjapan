# K-JAPAN Dashboard 🇯🇵

일본 여행을 위한 실시간 대시보드 웹 애플리케이션

## ✨ 주요 기능

- 🗺️ **Google Maps 통합**: 방문 장소 관리 및 대중교통 경로 찾기
- 📅 **여행 일정**: Google Calendar 실시간 연동
- ✅ **체크리스트**: 할 일 관리 (3인 체크 시스템)
- 💰 **정산 현황**: 실시간 비용 계산 및 정산 관리
- 📄 **여행 문서**: Google Drive 연동으로 항공권, 티켓 통합 관리

## 🚀 빠른 시작

자세한 설치 및 실행 방법은 [quickstart.md](./quickstart.md)를 참고하세요.

```bash
# 패키지 설치
npm install

# .env 파일 생성 및 Google Maps API 키 설정

# 서버 실행
npm start
```

## 🛠️ 기술 스택

- **Frontend**: HTML5, TailwindCSS, Vanilla JavaScript
- **Backend**: Node.js, Express
- **APIs**: Google Maps API, Google Sheets API, Google Drive API, Google Calendar API

## 📸 스크린샷

- 메인 대시보드: http://localhost:3000
- 여행 문서: http://localhost:3000/docs.html
